## lightfall-server

A server side only mod that allow joining a forge server through a lightfall proxy.

| Version | Support |
|---------|---------|
| 1.16+   | maybe   |
| 1.18.2  | yes     |
| 1.19.*  | planned |

[lightfall](https://github.com/ArclightPowered/lightfall) is a waterfall fork with modern forge support.
